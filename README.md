# Gardens

**MLS encrypted messaging for communities.**

Where privacy meets community — end-to-end encrypted group conversations for affinity networks.

![Gardens — Secure Community Protocol](./public/og.png)

## What is this

Gardens is a marketing/waitlist site for a secure group messaging platform built on the MLS (Message Layer Security) protocol with AES-256 encryption. Think Signal-level security, built for communities.

- **Protocol:** MLS v1.0
- **Encryption:** AES-256
- **Status:** Beta 2026

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Getting started

```bash
pnpm install
pnpm dev
```
